Post
In this base class, the Post operation is essentially a no-op. It returns a materialized copy of the object (which is meaningless unless the materializer implements some logic that does something to it), but fulfills the JSONAPI requirement that the POST operation return the POSTed object. It should probably be overridden in any implementation.
Put
Similar to Post, this method doesn't do much. It calls MaterializeUpdateAsync() on the input and returns it. It should probably always be overridden.
